## Repositório Aberto de TCC

Repositório aberto para compartilhamento de Trabalhos de Conclusão de Curso.

---

### Acesse o site
[https://repositorioaberto.xo.je](https://repositorioaberto.xo.je)

---

### Funcionalidades

**Desktop**
- Login com Google
- Dashboard estilo GitHub
- Edição de perfil
- Publicação via Google Forms
- Avaliações e discussões via GitHub Issues
- Visibilidade: público, privado, link restrito
- Modo escuro

**Mobile**
- Aplicativo PWA instalável
- Modo leitura otimizado
- Gamificação com conquistas
- Estatísticas de leitura
- Sincronização com desktop

---

### Tecnologias
HTML5, CSS3, JavaScript, Google Identity Services, GitHub Issues, PWA
